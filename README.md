# JD-Data-Code
This repository contains the Stata code and data for the master thesis "From Inventory Positioning to Delivery Promises: Empirical Evidence on Algorithmic Compensation in JD.com's Fulfillment Network" submitted to Rotterdam School of Management, Erasmus University (June 2026)

The analysis is based on the JD.com transaction-level dataset from the 2020 MSOM Data Driven Research Challenge (Shen et al., 2020), covering 549,989 order-item records across one anonymous consumable product category for March 2018. The dataset is publicly available for academic research at https://pubsonline.informs.org/doi/10.1287/msom.2020.0900 
The raw data files used are:

JD_order_data.csv
JD_delivery_data.csv
JD_user_data.csv
JD_sku_data.csv
JD_inventory_data.csv
JD_network_data.csv
